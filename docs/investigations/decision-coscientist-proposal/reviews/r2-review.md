## 维度一：novelty 区分度 — **4/5**

### 理由（真实优点）
- 收窄后的「三重限定组合」主张**站得住**：(i) 搜索对象为自然语言决策策略（非数值参数向量）、(ii) 适应度为对历史外部真值的校准度（非对同一仿真内部目标的距离）、(iii) 由「锦标赛+进化+元评审」生成性闭环驱动（非单点参数优化）。与 ABM 参数校准谱系的正面区分在文献层面成立——已核验 Stonedahl & Wilensky（MABS 2010/LNCS，pp. 61–75）与 Grazzini & Richiardi（JEDC 51:148–165, 2015）原文确实存在且确为参数空间标定/模拟矩估计，三者单独看确有祖先、组合处确无占位的表述方式是诚实且可辩护的。
- 与 LEAR 的区分**经原文核对准确**：LEAR 的 tournament selection 确为 GA 亲本采样算子（算法第 14 行 "Choose m parents using tournament selection"，正文明确其为保留多样性的亲本选择），适应度为采集环境内部计数，无 Elo、无配对排名、无外部真值、无 judge。v1.1 对 LEAR 的表述（2.3(a)）与原文一致。
- 与 ADAS/AFlow/EvoAgent/OPRO 的「agent 构造（代码/提示/工作流）× 任务分数」vs「决策策略语义层 × 历史校准度」区分成立（四篇均已核验存在）。
- 与 arXiv 2509.21868 的「制度嵌入 vs 自动化决策优化」互补定位经原文核对准确（~13,000 agents、16 个月、影响志愿者培训与疏散流程、validation filter、enabling conditions 均有原文支撑）；SLALOM 定位亦准确（stopped clock、gates+DTW、workshop、无灾害实证），且报告已用轨迹保真对照（5.4）正面承接其批评——这是教科书式的「引批评者入设计」做法。
- self-play 谱系（fictitious play→PSRO→league→α-Rank/Nash averaging→spinning tops）→「judge 噪声决定进化天花板」的理论嫁接是本案最有辨识度的理论落点；结算驱动校准（Halawi、Prophet Arena）向多智能体推演的扩展主张经核验无直接占位。

### 问题
- **【严重 N-a】两条先验线缺席，IJCAI/AAAI 评审人会立即提出：**
  1. **应急仿真优化（simulation-optimization / OR 谱系）**：用元启发式（GA 等）在离散事件/ABM 仿真上优化疏散与应急资源方案的运筹学文献已积累二十年以上（OptQuest 类仿真优化器、疏散仿真优化等）。该线与本案的差异仅在「策略载体为自然语言+LLM 角色扮演推演」与「适应度含历史校准分量」两点，报告全文未提及。做仿真/OR 的审稿人会视其为最直接先验。
  2. **LLM 提示/程序自动优化谱系（DSPy/MIPRO、Promptbreeder、EvoPrompt、TextGrad 等）**：自然语言对象的闭环自动优化 + 数据指标适应度，是三重限定中 (i)+(iii) 的近邻；当适应度取历史数据/已结算结果上的指标时，(ii) 亦被部分逼近。注意 LEAR 原文的参考文献中即含 Promptbreeder——该谱系与本案的相邻性有硬证据，报告只引 OPRO 而未覆盖此线，「NL 对象 × 闭环 × 数据指标适应度」的最强先验竞争者缺席。
- **【一般 N-b】** ABM 实时数据同化/状态估计线（粒子滤波同化 crowd ABM 等，Malleson、Ward 等）未切割。属状态估计而非策略搜索，一句话即可切开，但方法学家会期待这句话。
- **【一般 N-c】** 「Minasny 是目前唯一的跨域移植先例」「本课题是该架构向决策科学的首次移植」的「唯一/首次」依赖检索覆盖度，且报告自己承认窗口期以月计——建议统一软化为「据我们所知/截至检索日」。
- **【一般 N-d】** 2.5 节未引 Epstein 的生成性社会科学（generative sufficiency / IGSS）。Axtell 1996 docking 已引，但该谱系源头文献缺席会被 ABM 背景的审稿人注意（SLALOM 原文就引了 Epstein IGSS）。

## 维度二：证据链效度 — **3/5**

### 理由（真实优点）
- 锚分离原则（因子入库即标注 fitness_pool/eval_pool、两池不相交、H1 只在评估锚上判定）是对「胜出是构造性产物」循环的**正确且可操作**的回应；且各臂同 judge、同扰动分布、同 token 预算，引擎与 judge 的系统偏差是相对较中的共享项——H1 的相对比较不受其污染，设计结构公平。
- H1 判定规则具体可证伪：逐项+互不为负、须赢 B1/B2 才归因机制、仅赢 C0 不构成支持。防止了稻草人式宣称。
- 校验臂**确实独立于 LLM**（GB 50151 规范下限 × 供给曲线 × 回归速率上界；NUREG/CR-7002 ETE 排队模型），且采用「下限闸门+区间」而非点估计的判异设计，在方法论上正确（规范值本就是设计下限）；伤亡维度无独立参照被主动声明降级，H4 效度主张限定在覆盖维度——切割干净。
- H3 汇总判定规则（≥8 条因子、≥60% supported 成立 / <40% 不成立 / 其间 inconclusive 并公布明细）可证伪；≥3 名专家两轮盲评且首轮隐去地名与结局，是好的去先验设计；按因子类型预注册匹配粒度（区间重叠≥50% / 事件级 ±12h / 分布形态级）具体。

### 问题
- **【严重 E-1】同案切分 + LLM 先验：H3「旁证」的解释力被实质性削弱。** 评估锚与适应度锚来自同两份事故调查报告，因子级「不相交」不保证信息独立：(a) 同案因子共享同一事件叙事，天然相关；(b) 更根本地，生成策略的 LLM 对这两起超大公共事件有训练先验，其对留出因子的「知悉」无法被池隔离阻断。五条污染控制管的是文本级复述（verbatim 数字剔除、±30% 扰动变体、语料稀薄锚）与泄露基线测量，**管不住知识级泄露**。后果不对称：H1 不受影响（B1 等基线共享同一先验，机制归因的相对比较免疫）；但 **H3 没有对照臂**——头部策略命中留出历史因子可用「模型先验回忆」完全解释，无需推演机制。泄漏基线只测量、无预注册的降级/作废阈值；也没有「LLM 直接作答（无推演）」对照来量化推演的增量贡献。
- **【严重 E-2】因子池分配规则未预注册，且 H3 阈值与因子库存存在张力。** 随机分配 → 相关因子跨池泄露；按因子族分配 → 评估池内某些族可能整体缺席，H3 的「无任一因子族被系统性 rejected」将无法检验。同时现状为古雷 9 条已编码（若部分入适应度池）、郑州尚未编码，而预注册阈值要求「两案合计 ≥8 条进入裁定」——阈值有写死在库存摸清之前的嫌疑，若事后发现不足 8 条，任何调整都构成预注册变更。
- **【严重 E-3】H4 的「judge 排名 × 校验臂参照排序相关显著」无法按现描述操作化。** 校验臂输出的是闸门/区间判异（标红/不标红），不产生细粒度排序；二值闸门与 judge 连续排名之间的「排序相关」缺少构造（例如按「超出闸门的距离」构造连续严重度分数再算 Kendall τ——或任何等价方案，但须写明）。叠加伤亡维度无独立参照，H1 双主指标之一的效度证据落空（已声明降级，但 H1 的伤亡结论强度因此受限）。
- **【一般 E-4】Youden J 准则从梯度 RL 到进化选择的迁移是假设，不应写成硬性通过标准。** Rad et al.（arXiv:2601.04411，已核验）的相变是对 GRPO 的 bandit 解析模型（replicator 流），选择机制为梯度 RL 而非锦标赛/进化；「J>0 噪声只拖慢收敛」对进化选择无定理保证。另「逐局误判率 ≪10%（self-play 实测阈值）」的表述**高于文献实际**：Plesner et al.（已核验）给出的是「≤15% 噪声下峰值精度损失 <2pp」的特定设定实证，并非通用 10% 阈值。建议降为「参照 RLVR 实证的启发阈值」并注明迁移假设。
- **【一般 E-5】** H2「上升幅度显著超过选择伪影对照曲线」的两曲线比较统计量未定义。

## 维度三：统计严谨性 — **4/5**

### 理由（真实优点）
- 功效分析数字**复核无误**：双侧 α=0.05、功效 0.8，按 n=2(z₀.₉₇₅+z₀.₈)²/δ² 计算，δ=0.3 → ≈174.5（报告称 175）、δ=0.2 → ≈392（报告称约 395，四舍五入口径差异内）。放弃被本团队自己证伪的 N=240 先例、并写入 N=6 强信号在 N=30 反转的实测教训，是罕见的统计诚实。
- 配对设计（同扰动种子跨条件配对）+ Wilcoxon 符号秩：方向正确（配对提高功效），非参数检验对非正态指标稳健；非配对用置换检验/Mann-Whitney U 亦恰当。
- 锚对照采用反转零假设 + 预注册等价界值（相对差异 ≤15%）+ bootstrap——正确执行了 Axtell et al. 1996 的药方，防「未能拒绝」冒充等价。
- token 预算口径（总 token 含 judge/生成/元评审、各臂一致）是公平比较的正确共同货币；G2 统计合法性（LLM 采样方差为零、条件独立性）被列为 W4 go/no-go 而非掩盖——处置正确，这是全案最大的存在性风险，管理方式是诚实且果断的。
- BH 校正、Cliff's δ + CI、模型版本钉死 + 漂移监测、拉丁超立方 + 固定种子，清单主体完整。

### 问题
- **【严重 S-1】预注册清单有实质缺口：** (a) 进化的种群规模/代数/每代配对对决数未预注册——α-Rank 排名基于成对胜率矩阵，样本量直接决定排名稳定性，且与 token 预算强联动；(b) BH 校正的「检验族」边界未定义（哪些 指标×条件对 构成一个校正族）；(c) H2 曲线比较统计量未定义（与 E-5 同）；(d) 等价界值「相对差异 ≤15%」的参照基准未写明（相对锚区间宽度？相对锚中点？相对推演均值？——不同基准结论不同）。
- **【一般 S-2】** G2 部分失败（iid 检验不通过但非全灭）时的降级检验路径未写：配对情形下 Wilcoxon 的合法性依赖条件独立性，降级方案（如基于聚类/块的稳健推断）应在预注册中备妥，而非届时临场选择（构成分析自由度）。
- **【一般 S-3】** α-Rank 在噪声成对胜率下的样本复杂度未估算；每代 N 场对决的最小可接受 N 应进预注册（可与 W4 成本试跑联动）。

## 维度四：工程可行性 — **3/5**

### 理由（真实优点）
- 已有零件经磁盘核查、非纸上谈兵：PolicySim 引擎（三场景八角色、约 1200 项测试）、`tournament.py`（679 行，BT+Elo+CI+自适应配对）、`calibration_lib.py`（Gold-H/M/L+漂移检测）、Factor Ledger 全套 schema/预注册 lock、古雷 9 因子试编码已完成且适配改动已列清单。
- 成本实测前移至 W4 + 明确 go/no-go + 预写回退条款（workshop 降级、与 Tsinghua 09 合并、负结果写方法短文）——风险管理成熟，防路径振荡的回退条款是预注册纪律的体现。
- 校验臂选型有监管级方法与历史标定先例（Katrina/Irma ETE、Goyang 反演、480 起罐火案例库），工程参照的可信性论证到位。

### 问题
- **【严重 F-1】W5–W9 五周工期偏紧且无缓冲。** 该窗口内含：策略级锦标赛编排（引擎语义从「模型×问题」改造为「策略×场景」）、选项池变异/进化算子、Multi-Judge **从零实现**（现仅空表头）、元评审回灌闭环、Gold 锚灾害域重建；并行还有 sim 侧三个可观测性缺口（初控-复燃两阶段状态机、罐区 1.5D 几何拓扑、伤亡伤/亡分离）——这三个是核心引擎改动，且是相关因子可裁定性的前置条件。按常规人效超出单人五周可行量；W13 主实验硬依赖 W12 锚协议就绪，链上任何滑动直接吞噬 W18–W20 写作期。
- **【严重 F-2】外部依赖无缓冲：** ≥3 名应急领域专家的两轮盲评招募无启动时间、无截止线、无备选方案（应急领域专家招募通常以周-月计）；跨底座模型复现算力预算悬置至 W4 之后；课题三知识库 API 为外部依赖。
- **【一般 F-3】** 郑州因子编码「估 1 人日」相对古雷试编码的实际投入（含盲评字段、来源分级、双挂钩判定）明显乐观；且 H3 需要两案合计 ≥8 条可裁定因子，编码量与预注册阈值挂钩（与 E-2 联动）。
- **【一般 F-4】** 若 W4 go/no-go 触发部分降级（如砍跨底座复现），16–20 周时间表无修订版；IJCAI 2027 投稿目标与降级路径的兼容性未说明。

## 维度五：诚实边界 — **5/5**

### 理由
- 八条边界逐条有对策且措辞克制：Elemento 式边界（只主张搜索效率与校准质量，不主张机器原创处置方案）、真值硬度（「旁证」措辞 + 正负双锚 + 留出锚 + 反转零假设）、复现≠解释（结论限后果分布层）、等终性（多锚点 + 锚分离 + 反事实机制检验）、信度≠效度（校验臂相关 + Youden J + 跨 judge/跨底座）、训练数据污染（五条控制）、内部主张切割（与 Tsinghua 09 三轴切割 + 回退合并条款）、单司法辖区（Buncefield 第三锚候选）。
- AI 生成来源（Chen 2026）显式标注并降为动机性参照、不承担承重论断；v1.0→v1.1 更正了 LEAR 表述与 Kwon 作者顺序；主动放弃被自己证伪的 N=240 先例并写入教训；校验臂覆盖缺口（伤亡维度）主动声明降级——未发现隐瞒的弱点。
- novelty 从「谱系中无先例」自我收窄为三重限定组合，正是对过宽主张的自我修正。
- 唯一接近瑕疵处：「首次移植」类措辞偏强（见 N-c），以及「≪10% self-play 实测阈值」的表述高于文献实际（见 E-4）——均为一般级，不构成隐瞒。
