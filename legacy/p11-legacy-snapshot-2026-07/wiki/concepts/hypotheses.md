# 研究假设 H1-H3

> **来源**：state/task_spec.md

---

| 编号 | 假设 | 检验方法 | 状态 |
|:--:|------|------|:--:|
| **H1** | 内心独白 Agent 的 RoleDNA Fidelity 高于无 think Agent | Mann-Whitney U | ⏳ |
| **H2** | 纯分析 Agent 产生更清晰但更少"角色特征"的决策 | 描述性对比 | ⏳ |
| **H3** | 角色一致性与涌现真实性正相关 | Spearman's ρ | ⏳ |

**成功标准**：
- Mode B Fidelity > Mode A by ≥ 0.5 (p<0.05)
- Think-tag 触发率 ≥ 85%
- Spearman's ρ ≥ 0.4
