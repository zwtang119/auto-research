# 涌现行为

> **定义**：多 Agent 交互中，个体行为聚合后产生的、无法从单个 Agent 行为直接预测的群体现象。

## 度量方法

- **策略多样性**：Shannon 熵——不同 Agent 策略分布
- **涌现关键词**：多个 Agent 独立提及但 prompt 中未给出的概念
- **涌现真实性**：Multi-Judge 评估群体交互是否呈现真实应急场景的典型模式
