# Codability Annotation Pack v0.1

本目录是一套可转发的人类标注包，用于 CDS4WorldCup2026 Plan B2 Codability Census。

## 文件说明

- `codability-calibration-20.xlsx`：校准试标，20 条，不进入最终统计。
- `codability-annotation-A-full300.xlsx`：主标注文件，300 条全量 reason。
- `codability-annotation-B-kappa100.xlsx`：第二标注文件，100 条，用于一致性检验。
- `codability-rubric-v0.1.md`：标注规则说明。
- `codability-codebook.csv`：字段和选项定义。
- `send-guidance.md`：发送给标注者的操作建议和消息模板。
- `questionnaire-fields.md`：如果改用问卷星/飞书/Google Forms，可复制的问题字段。
- `private/codability-id-map-private.csv`：私有映射，绝对不要发送给标注者。

## 标注原则

标注者只判断 reason 是否“原则上可审计”，不判断真假，不查资料，不根据球队常识补脑。
