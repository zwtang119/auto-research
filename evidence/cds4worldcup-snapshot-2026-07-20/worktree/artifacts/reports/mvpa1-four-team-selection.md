# MVP-A1-QA 四队深描质检样本选择

> **状态**: draft
> **日期**: 2026-06-11

## 目的

MVP-A1 的范围是 48 队全覆盖薄切片路径卡。本文件只选择 4 支代表球队作为 **MVP-A1-QA 深描质检样本**，用于验证模板能否从薄切片升级为深描。

四队必须覆盖不同 `path_signals`，避免样本都集中在热门队。最终 `path_type` 不能由本文件预设，必须等待 48 队 signal matrix 和 Plan B 数据分析后再派生。

## 推荐组合

### 1. 顶级热门：西班牙

候选 path signals：

- `dominant_favorite`
- `youth_surge`

选择理由：

- Kimi 300 Agent 热力榜中西班牙是最高或接近最高的热门之一。
- 适合测试“强队不是只看概率，而是看哪些条件会让强队失效”。
- 可观察阻力包括：年轻核心伤病、连续高压状态疲劳、传控高控球低 xG、淘汰赛被低位防守拖入点球。

### 2. 传统强队：阿根廷

候选 path signals：

- `veteran_tournament_craft`
- `penalty_survival`

选择理由：

- 有卫冕、梅西最后一舞、点球与门将叙事，适合区分可判定因子和不可判定 Marginalia。
- 可观察阻力包括：年龄结构、卫冕魔咒、关键球员体能、淘汰赛路径。

### 3. 中强队：葡萄牙

候选 path signals：

- `bracket_leverage`
- `superstar_spike`
- `youth_surge`

选择理由：

- 兼具 C 罗叙事和新生代阵容，容易观察“老将精神图腾”和“阵容硬实力”如何分离。
- 适合测试路径套利：若小组和半区友好，葡萄牙可能从中强队转为现实候选。

### 4. 黑马：摩洛哥

候选 path signals：

- `low_variance_defense`
- `penalty_survival`
- `favorite_collapse_required`

选择理由：

- Kimi reason 中多次出现摩洛哥，且包含 FIFA 排名、非洲杯、U20 世冠、2022 四强、门将等多类可判定与不可判定材料。
- 适合验证“黑马夺冠到底需要多少个条件同时成立”。

## 验收标准

四队深描质检完成后，应能回答：

- 四队的夺冠故事是否显著不同。
- 每队至少 3 个阻力是否具体。
- 每队至少 3 个黑天鹅助力是否明确。
- 每队至少 3 个 Factor Ledger candidates 是否可判定。
- 每队至少 3 条 Marginalia notes 是否保留了不可判定但有价值的叙事。
- 薄切片模板升级为深描卡时，是否需要新增字段。
- 候选 `path_signals` 是否足以支持后续数据派生 Path Type。
