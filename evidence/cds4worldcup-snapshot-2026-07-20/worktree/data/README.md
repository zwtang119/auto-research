# data/ — 数据文件

本目录存放项目使用的数据文件。

## 目录约定

```
data/
├── raw/                # 原始数据（.gitignore 排除，不上库）
├── processed/          # 清洗后数据（上库，用于复现）
└── README.md           # 数据来源说明
```

## 数据来源

| 数据集 | 来源 | 更新频率 | 状态 |
|--------|------|----------|------|
| FIFA 赛程 | FIFA 官方 | 赛前确定 | 📋 待获取 |
| 球队排名 | FIFA Ranking | 月度 | 📋 待获取 |
| 历史比赛结果 | 公开数据集 | 一次性 | 📋 待获取 |

## 规范

- `raw/` 不上库（在 .gitignore 中），保留原始数据完整性
- `processed/` 上库，确保结果可复现
- 每个数据文件必须在本文件中登记来源和时间
- 大文件（>10MB）使用 Git LFS 或外部存储
